# react-native-lite-camera

A lightweight, fast, and easy-to-use React Native Camera module.

## Installation


```sh
npm install react-native-lite-camera
```


## Usage


```js
import { multiply } from 'react-native-lite-camera';

// ...

const result = multiply(3, 7);
```
